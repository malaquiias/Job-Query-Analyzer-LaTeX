\section{Descrição do Dataset}%
\label{sec:DescriodoDataset}%
A análise foi realizada utilizando um conjunto de dados público da plataforma Kaggle.\newline%
%
\\%
\textbf{Arquivo:}%
 engenheiro\_de\_dados\_6k.csv 
%
\\%
\textbf{Conteúdo:}%
 O dataset contém 1.614 vagas para a posição de Engenheiro de Dados (Data Engineer) no Brasil (LinkedIn, 2023). Para esta análise, os 'documentos' são os textos da coluna 'job\_description'.

%
\section{Metodologia da Análise}%
\label{sec:MetodologiadaAnlise}%
O objetivo deste trabalho é comparar a aderência de dois mercados de trabalho (São Paulo e Remoto) a um perfil de vaga ideal, definido pelo usuário (a 'query').%
\par%
\subsection{Definição da Query e Documentos}%
\label{subsec:DefiniodaQueryeDocumentos}%
\textbf{A Query (Perfil Ideal):\newline%
}%
\\%
O usuário forneceu o seguinte texto em inglês descrevendo seu perfil ideal:\newline%
%
\begin{it}%
data enginieer with experience in python and sql%
\end{it}%
\\ \par%
\textbf{Os Documentos (Os Grupos):\newline%
}%
\\%
Os documentos da base foram divididos em dois grupos:\newline%
%
\\%
{-} \textbackslash{}textbf\{Grupo 1 (São Paulo):\} 756 documentos.\newline%
%
\\%
{-} \textbackslash{}textbf\{Grupo 2 (Remotas):\} 180 documentos.\newline%

%
\subsection{Processamento (Álgebra Linear)}%
\label{subsec:Processamento(lgebraLinear)}%
1. %
\textbf{TF{-}IDF:}%
 A Query e todos os Documentos de um grupo são transformados em vetores numéricos que representam a importância de cada palavra{-}chave.\newline%
%
\\%
2. %
\textbf{Similaridade de Cosseno (Análise de Ângulos):}%
 É calculada a similaridade (cosseno do ângulo) entre o vetor da Query e *cada um* dos vetores de documentos do grupo. Um ângulo de 0° (cosseno = 1) significa que a vaga é idêntica à query.
